# SmartBrainApp-FrontEndPart

This SmartBrain is a Face Recognition App created by using ReactJS


## To run this app:
### 1st step - 
Clone this repo,
### 2nd step - 
Run npm install and 
### 3rd step -
Run npm start.

**You must get your own API key from [here](https://clarifai.com/)**
